Storage Connectors

Storage connectors enable deepstream to store data in a database. This allows to use the database's query capabilities and to persist data when deepstream is used with an in-memory only cache like Memcached. Storing data in a database happens outside of the critical path and doesn't affect update performance.

  • rethinkdb storage connector

    deepstream.io-storage-rethinkdb
    0.2.6

    0.2.6
